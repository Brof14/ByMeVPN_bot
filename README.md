# ByMeVPN Bot

Асинхронный Telegram-бот для продажи VPN-подписок.

## Стек
- **aiogram 3** — фреймворк бота
- **aiosqlite** — SQLite с WAL-режимом
- **py3xui** — управление 3x-UI панелью
- **httpx** — ЮKassa API

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать .env
cp .env.example .env
# — Заполнить BOT_TOKEN, XUI_HOST, XUI_USERNAME, XUI_PASSWORD, INBOUND_ID
# — Заполнить REALITY_HOST, REALITY_PBK и остальные VLESS-параметры

# 3. Запустить
python main.py
```

## Структура

```
main.py            — точка входа, регистрация роутеров
config.py          — все настройки из .env
database.py        — все операции с БД (aiosqlite)
xui.py             — работа с 3x-UI через py3xui
subscription.py    — ядро: запрос имени конфига → создание ключа → выдача
payments.py        — создание платежей ЮKassa
keyboards.py       — все клавиатуры
utils.py           — хелперы (форматирование, send_or_edit)
states.py          — FSM-состояния
notifications.py   — фоновые уведомления об истечении

handlers/
  start.py         — /start, главное меню, пробный период, back_to_menu
  buy.py           — выбор тарифа, оплата Stars/ЮKassa, обработка платежа
  keys.py          — «Мои ключи», продление, удаление
  partner.py       — партнёрская программа
  guide.py         — инструкция подключения (iOS/Android/Windows/macOS/Linux)
  admin.py         — /admin, статистика, рассылка
  fallback.py      — обработка неизвестных сообщений (должен быть последним)
```

## Ключевые решения

### Пробный период
- **Бесплатно** (0 ₽), 3 дня
- После нажатия «Попробовать бесплатно 3 дня» сразу спрашивает имя конфига
- После ввода имени — создаёт клиента в 3x-UI и выдаёт VLESS-ключ

### Telegram Stars
- `stars_amount = price_rub` (1:1, без конверсии)
- Это сделано намеренно для покрытия высоких комиссий Telegram

### Выдача ключей
1. После любой активации (пробник / Stars / ЮKassa) → FSM-состояние `waiting_name`
2. Пользователь вводит название (например: «iPhone», «Работа»)
3. Бот создаёт клиента в 3x-UI через `py3xui.AsyncApi`
4. Генерирует VLESS-ссылку с параметрами из `.env`
5. Сохраняет в БД и отправляет ключ пользователю

### Мои ключи
Каждый ключ отображается с кнопками:
- **🔄 Продлить** — переход к выбору тарифа
- **🗑 Удалить** — удаление из 3x-UI и БД (с подтверждением)

### Рефералка
- Реферер получает уведомление, когда реферал совершает первую оплату
- Бонус (+3 дня) выдаётся через ту же FSM-цепочку с запросом имени конфига

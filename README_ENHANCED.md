- Python 3.8+

- Доступ к XUI панели
- Настроенные платежные системы

## 📦 Установка и запуск

### 1. Клонируйте и установите зависимости:
```bash
cd /root/byme_vpn_bot
pip install -r requirements.txt
```

### 2. Настройте переменные окружения в `.env`:
```bash
BOT_TOKEN=your_bot_token
ADMIN_ID=your_admin_id
REF_BONUS_DAYS=7
SUPPORT_USERNAME=ByMeVPN_support
MENU_PHOTO=path_to_menu_photo.jpg

# XUI настройки
XUI_HOST=your_xui_host
XUI_BASE_PATH=/your/path
XUI_USERNAME=your_username
XUI_PASSWORD=your_password
INBOUND_ID=1

# Reality настройки
REALITY_HOST=your_reality_host
REALITY_PORT=443
REALITY_SNI=your_sni
REALITY_FP=your_fingerprint
REALITY_PBK=your_public_key
REALITY_SID=your_session_id

# Платежи
YOOKASSA_SHOP_ID=your_shop_id
YOOKASSA_SECRET_KEY=your_secret_key
CRYPTOBOT_TOKEN=your_cryptobot_token
STAR_RATE_RUB=0.1
```

### 3. Запустите бота:
```bash
python main.py
```

## 🎨 Новые функции и как ими пользоваться

### 📊 Динамический счётчик пользователей
- Автоматически показывается в приветствии
- Считает уникальных пользователей за текущий месяц
- Обновляется в реальном времени

### 🎨 Цветные кнопки
- **🟢 Зелёные** - покупка, тарифы, пробный период
- **🟡 Золотые** - акции, турниры, автопродление
- **🔵 Синие** - поддержка, помощь, инструкции
- **⚪️ Белые** - юридические документы
- **⚫️ Тёмные** - навигация, "назад"

### 🤝 Улучшенная реферальная программа
1. Нажмите "Партнёрская программа"
2. Видите свою статистику:
   - Сколько пригласили
   - Сколько оплатили
   - Сколько бонусов получили
3. Нажмите "Мои рефералы" - подробный список
4. Поделитесь ссылкой - красивый формат

### 💳 Умная история платежей
- Показывается только если у вас есть оплата
- Эмодзи для способов оплаты
- Общая сумма потраченных средств

### 🔔 Улучшенные уведомления о продлении
- **За 3 дня** - ⏰ напоминание
- **За 1 день** - ⚠️ срочное уведомление  
- **В день истечения** - 🚨 критическое
- **Автопродление** - автоматические счета

### 📦 Экспорт ключей
- Настройки → Экспортировать ключи
- Файл .txt с полной информацией
- Даты создания, окончания, серверы

### 🏆 Турниры (система готова)
- Админ может создавать турниры
- Условия: пригласи X друзей, оплати Y раз
- Награды: дни VPN
- Автоматический подсчёт прогресса

### 🔧 Автоматическая проверка ключей
- Каждые 6 часов проверка работоспособности
- Автоматическая замена неработающих ключей
- Уведомления пользователей о замене

## 📈 Мониторинг

### Что проверять:
1. **Работа бота** - логи, ответы на команды
2. **Платежи** - поступление средств, выдача ключей
3. **Уведомления** - отправка напоминаний
4. **Серверы** - здоровье ключей, замены

### Логи важных событий:
- Выдача подписок
- Платежи
- Замены ключей
- Ошибки

## 🚀 Запуск в продакшене

### Рекомендации:
1. **Используйте PM2** для управления процессами
2. **Настройте мониторинг** логов
3. **Резервное копирование** базы данных
4. **Мониторинг** здоровья серверов XUI

### Пример запуска с PM2:
```bash
pm2 start main.py --name "bymevpn-bot"
pm2 save
pm2 startup
```
